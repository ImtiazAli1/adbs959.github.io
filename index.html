My first programming project, concluding the 2 week Programming Bootcamp at City (2020). This Processing project demonstrates a comprehensive understanding of the use of arrays, loops, and randomness. I have created some visual art, where a set number of shapes will be set to spin around the page getting smaller in the process. Once it reaches the middles it will 'explose' into balls that will bounce off the walls at all random speeds and sizes, untill the program comes to an end by a set timer.
